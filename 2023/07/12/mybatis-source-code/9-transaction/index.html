9-transaction

Transaction

​ Transaction包装了一个数据库连接，处理连接的生命周期，包括:创建、准备、提交/回滚和关闭。在Mybatis中有两种事物类型（也就是 type=”[JDBC|MANAGED]”）

TransactionFactory

​ 事务工厂

JdbcTransaction

​ JdbcTransaction直接使用JDBC提交和回滚功能。它依赖从数据源检索的连接来管理事务的范围。

​ 它会延迟连接获取知道getConnection()方法被调用。当autocommit打开的时候忽略commit或rollback请求。

JdbcTransactionFactory

​ JdbcTransaction工厂

ManagedTransaction

​ ManagedTransaction让容器管理事务的整个生命周期。和JdbcTransaction一样，延迟获取数据库连接直到getConnection()方法被调用

​ 默认情况下，它会关闭连接，但可以将 closeConnection 属性设置为 false不关闭连接。

​ 如果使用mybatis-spring的话，不需要配置transactionManager ,因为mybatis-spring覆盖了mybatis里的逻辑。

ManagedTransactionFactory

​ 托管事务工厂,默认情况下它会关闭连接。
